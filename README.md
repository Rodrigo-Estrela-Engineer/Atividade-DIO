# Atividade-DIO


Este repositório foi criado para a entrega do exercício proposto pela plataforma DIO. A atividade consiste no preenchimento do Mapa de Habilidades DIO, contemplando corretamente as três etapas solicitadas, conforme as orientações do curso.
Após o preenchimento do material, o conteúdo foi organizado e disponibilizado neste repositório do GitHub, com o objetivo de facilitar o acesso e a avaliação. Por fim, o link deste repositório foi enviado na plataforma da DIO como forma de conclusão da atividade.
